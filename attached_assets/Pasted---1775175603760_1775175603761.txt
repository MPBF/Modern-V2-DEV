مذكرة تحديث وتطوير نظام "المساعد الذكي للمصنع"
يرجى مراجعة الكود الحالي وتطبيق التحسينات التالية لرفع كفاءة الأداء، الأمان، وتجربة المستخدم:

1. تفعيل خاصية الـ Streaming في OpenAI (تحسين الاستجابة)
بدلاً من انتظار الرد الكامل من OpenAI (الذي قد يستغرق ثوانٍ طويلة)، يجب تحويل الاستجابة إلى Stream.

المطلوب: تعديل طلب openai.chat.completions.create ليتضمن stream: true.

الهدف: إظهار الإجابة للمستخدم كلمة بكلمة فور توليدها، مما يقلل "زمن الانتظار المتصور".

2. تأمين استعلامات قاعدة البيانات (SQL Injection Protection)
في أداة execute_database_query حالياً يتم استخدام sql.raw.

المطلوب: استخدام Parameterized Queries أو التحقق الصارم من المدخلات.

توصية أمنية: يفضل ربط الوكيل بـ Database Role بصلاحيات "القراءة فقط" (ReadOnly) للعمليات الاستعلامية، وصلاحيات محدودة جداً للجداول المطلوبة فقط عند الإضافة.

3. معالجة الذاكرة عند التعامل مع الملفات (Memory Management)
تستخدم الدوال الحالية Buffer.concat لتجميع ملفات PDF و Excel في الذاكرة.

المطلوب: الانتقال إلى نظام Streams عند إنشاء المستندات ورفعها إلى Object Storage.

الهدف: منع حدوث Memory Leak أو انهيار التطبيق عند طلب تقارير ضخمة تحتوي على آلاف السطور.

4. توحيد معالجة اللغة العربية (Arabic RTL Logic)
يوجد تكرار لمنطق ArabicReshaper و bidi في أماكن مختلفة.

المطلوب: إنشاء Utility Class موحد يسمى ArabicTextService يتعامل مع:

قلب النص (RTL).

التشكيل (Reshaping).

معالجة الأرقام والرموز داخل النصوص العربية.

الهدف: ضمان ظهور التقارير (PDF, Word) بشكل متناسق 100% دون أخطاء في قلب الحروف.

5. تحسين نظام الـ SSE (Keep-Alive & Cleanup)
المطلوب: التأكد من إغلاق كافة الـ intervals (مثل الـ Ping) والـ streams فور انقطاع اتصال العميل (req.on("close")).

الهدف: منع استهلاك موارد الخادم (CPU/RAM) بعد مغادرة المستخدم للدردشة.

6. تحديث إصدار الموديل (Model Versioning)
الملاحظة: الكود يشير إلى gpt-4.1.

المطلوب: التأكد من استخدام أسماء الموديلات الرسمية المستقرة مثل gpt-4o أو gpt-4-turbo لضمان استمرارية الخدمة.

💡 مثال مقترح لتطوير دالة الشات (Streaming Version):
TypeScript
// هذا الجزء يوضح كيف يجب أن يتحول الرد إلى Stream
const stream = await openai.chat.completions.create({
  model: "gpt-4o",
  messages: chatMessages,
  tools,
  stream: true, // تفعيل التدفق
});

for await (const chunk of stream) {
  const content = chunk.choices[0]?.delta?.content || "";
  if (content) {
    res.write(`data: ${JSON.stringify({ content })}\n\n`);
  }
}
ملاحظة للمطور: يرجى الحفاظ على نظام pg_advisory_xact_lock المستخدم في إنشاء الطلبات لأنه ممتاز جداً لمنع تكرار الأرقام التسلسلية.
